# 🦷 Fila de Pacientes

Projeto desenvolvido durante meus estudos de JavaScript, simulando o gerenciamento de uma fila de atendimento em um consultório odontológico.

## 🚀 Funcionalidades

- Adicionar pacientes ao final da fila
- Inserir pacientes com prioridade
- Atender o primeiro paciente da fila
- Exibir o paciente em atendimento
- Atualizar automaticamente a lista de espera
- Validar o campo de nome

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript

## 📚 Conceitos praticados

- Arrays
- Métodos `push()`, `unshift()` e `shift()`
- Eventos `submit` e `click`
- Manipulação do DOM
- Estruturas de repetição com `for`
- Funções
- Validação de dados
- Atualização dinâmica da página
